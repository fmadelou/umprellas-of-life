# umprellas-of-life
Ένα συνεργατικό σενάριο νηπιαγωγείων που συνδέει τα δικαιώματα του παιδιού με τη ρομποτική και τον ανοιχτό προγραμματισμό. Τα παιδιά δημιουργούν «Σταθμούς Δικαιωμάτων» χρησιμοποιώντας εξοπλισμό ρομποτικής , προγραμματισμού και ανοιχτού κώδικα, μαθαίνοντας ότι η τεχνολογία μπορεί να είναι σύμμαχος προστασίας των δικαιωμάτων και του κοινού καλού
